# Нумерованный список

Что бы создать нумерованный список , необходимо перед словом, цифрой, текстом поставить номер с точкой, например :
1. номер 1
2. номер 2
3. номер 3
    1. вложеный
    2. вложеный

Для ненумерованных списков используется *, или -, или +, например:
* номер
+ номер 
- номер
